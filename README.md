# Zomato Restaurants Data Analysis

## Project Overview
This project focuses on analyzing the Zomato restaurant dataset to extract meaningful business insights using Python, SQL, and data visualization techniques. The objective is to understand customer preferences, restaurant distribution, pricing trends, and service availability.

---

## Tools and Technologies
- Python
- Pandas, NumPy
- Matplotlib, Seaborn
- SQLite (for SQL queries)
- Jupyter Notebook

---

## Dataset Information
The dataset contains detailed information about restaurants including:
- Restaurant name and location
- Cuisines offered
- Average cost for two people
- Ratings and votes
- Online delivery and table booking availability

---

## Data Analysis Process

### 1. Data Understanding
- Explored dataset structure and column names
- Identified data types and key variables

### 2. Data Cleaning and Preprocessing
- Renamed columns for better readability
- Handled missing values
- Converted data types where necessary
- Removed inconsistencies in data

### 3. Exploratory Data Analysis (EDA)
- Analyzed restaurant distribution across cities
- Identified popular cuisines
- Studied cost trends for two people
- Examined rating distributions

### 4. Data Visualization
- Bar charts for city-wise analysis
- Count plots for categorical variables
- Histograms for rating and cost distribution

---

## Key Performance Indicators (KPIs)
- Total number of restaurants
- Average restaurant rating
- Top cities with highest number of restaurants
- Availability of online delivery services
- Distribution of pricing (cost for two)
- Most popular cuisines

---

## SQL Analysis
SQL queries were executed using SQLite to extract insights:
- Top 5 cities by restaurant count
- Distribution of online delivery availability
- Average rating of restaurants

---

## Project Structure
- `notebook.ipynb` – Main analysis and visualizations  
- `zomato.csv` – Raw dataset  
- `cleaned_zomato.csv` – Processed dataset  
- `queries.sql` – SQL queries used for analysis  
- `zomato.db` – SQLite database file  

---

## Key Insights
- Certain cities dominate in terms of restaurant density
- Online delivery plays a significant role in accessibility
- Ratings vary based on multiple factors such as cost and services
- Customer engagement differs across locations

---

## Conclusion
This project demonstrates how data analysis techniques can be applied to real-world datasets to generate actionable insights. It highlights the importance of data preprocessing, visualization, and SQL querying in making informed business decisions.
